# 要件定義書 — 時間外・旅費精算書アプリ

最終更新: 2026年5月4日

---

## 1. プロジェクト概要

### 成果目標
社員が1回入力するだけで、規程に準拠した残業時間・深夜作業手当の自動計算と、旬報（旅費交通費等精算書）の両方を出力できる。

### 成功指標

**定量的指標**
- 残業手当の計算ミスをゼロにする
- 報告書作成時間を 5分以内 に短縮
- 残業報告書と旬報の二重入力を解消（1回入力で両方出力）

**定性的指標**
- 規程（時間外手当基準）に完全準拠した計算が行われる
- 現場・社内どちらの作業区分にも対応できる
- 旅費・日当の記録から旬報Excelを自動生成できる
- 報告者が計算方法を知らなくても正しい手当額を把握できる

---

## 2. 規程準拠ルール

出典: `時間外手当基準（原紙）.xlsx`

### 残業開始時間
- **常用（平日）**: 勤務開始時間から **9時間30分後** から時間外手当を支給
- **休出（休日）**: 勤務開始時間から全時間が残業対象
- **有給・代休**: 残業計算なし

### 休憩時間（規程定義）

| 時間帯 | 休憩 |
|--------|------|
| 08:30 〜 09:00 | 0.5h |
| 12:00 〜 13:00 | 1.0h |
| 15:00 〜 15:30 | 0.5h |
| 18:00 〜 18:30 | 0.5h |
| 22:00 〜 22:30 | 0.5h |
| 02:00 〜 03:00 | 1.0h |
| 06:00 〜 06:30 | 0.5h |

### 時間外手当の種別

| 種別 | 時間帯 | 金額 |
|------|--------|------|
| 普通残業手当 | 残業時間のうち 6:00〜22:00 | 給与計算による |
| 深夜残業手当 | 残業時間のうち 22:00〜翌6:00 | 給与計算による |
| 深夜作業手当 | 18:00〜翌6:00 の常用勤務が5時間以上の場合 | **一律 3,000円** |

**注意事項（規程より）**
- 深夜作業手当と同時間帯の時間外作業手当は重複して支給しない
- 時間外時間は 0.5時間単位で集計（四捨五入）

## 5. 計算ロジック詳細

### 残業時間算出

```
勤務区分が有給・代休の場合: 残業 = 0（計算スキップ）

overtimeStart（常用）= 勤務開始時刻 + 570分（9時間30分）
overtimeStart（休出）= 勤務開始時刻

普通残業 = 残業時間帯 ∩ [6:00, 22:00] の実働時間（休憩除く）
深夜残業 = 残業時間帯 ∩ [22:00, 翌6:00] の実働時間（休憩除く）
```

### 深夜作業手当

```
常用勤務の深夜時間帯在籍 = [勤務開始, overtimeStart] ∩ [18:00, 翌6:00] の実働時間
深夜作業手当 = 常用勤務の深夜時間帯在籍 ≥ 300分（5時間）? 3,000円 : 0円
休出の場合は常に 0円
```

### 時間の丸め

0.5時間単位（四捨五入）: `Math.round(minutes / 30) * 0.5`

### 交通費

往復選択時: 入力金額 × 2 で登録

## 10. Excel出力仕様

### 残業報告書（個人別シート）

- ヘッダー: 時間外作業報告書、令和○年○月○日、報告者氏名、社員番号
- 明細: 区分・勤務区分・日付・オーダー №・件名・開始・終了・普通残業H・深夜残業H・深夜作業手当
- 合計行

### 残業報告書（一覧シート）

- 全登録者の明細一覧（同カラム構成）
- 合計行

### 旬報（旅費交通費等精算書シート）

- ヘッダー: 旅費交通費等精算書、令和○年○月○旬、氏名・社員番号・所属
- 明細: 日付・オーダー №・件名・勤務区分・開始・終了・日当・交通機関・区間・交通費・宿泊費・小計・備考
- 合計行
- オーダー別集計: オーダー番号・件名・日数・交通費・日当・勤務原価・合計

### ファイル名

- 残業報告書: `時間外作業報告書_YYYYMMDD.xlsx`
- 旬報: `旬報_YYYY年M月○旬_氏名.xlsx`

## 11. 提出フロー（業務）

**残業報告書**
```
報告者が入力・Excel出力
  ↓
直近上司 確認・承認
  ↓
所属部長 承認
  ↓
事務 受理
```
提出期限: 各旬報〆日から3営業日以内

**旬報（旅費交通費等精算書）**
```
報告者が入力・Excel出力（残業報告書と同時）
  ↓
上長 確認・承認
  ↓
事務 受理・精算処理
```
