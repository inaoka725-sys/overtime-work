## 実装済み機能

### 残業報告書タブ
- [x] 作業入力フォーム（日付・氏名・社員番号・オーダー・件名）
- [x] 勤務区分（常用 / 休出 / 有給 / 代休）
- [x] 区分（現場作業 / 社内作業）
- [x] 作業時間（30分刻みセレクト、日またぎ対応）
- [x] 帰社後作業（現場→帰社継続、全時間残業対象）
- [x] 残業時間リアルタイム自動計算（普通残業H・深夜残業H）
- [x] 深夜作業手当自動判定（18:00〜翌6:00の常用勤務5h以上で3,000円）
- [x] 複数エントリー管理（登録・削除・日付ソート）
- [x] 合計サマリー表示（普通残業合計・深夜残業合計・深夜作業手当合計）
- [x] 残業報告書 Excel 出力（個人別シート + 一覧シート）
- [x] LocalStorage によるデータ永続化

### 旅費入力フィールド（旬報連携）
- [x] 日当（0円 / 150円 / 300円）
- [x] 交通費（交通機関・区間・金額・片道/往復選択）
- [x] 宿泊費
- [x] 備考

### 旬報タブ
- [x] 年月・旬（上旬/中旬/下旬/全期間）フィルター
- [x] 旅費明細一覧表示
- [x] オーダー別集計（日数・交通費・日当・勤務原価）
- [x] 旬報 Excel 出力（旅費交通費等精算書フォーマット）

### 設定タブ
- [x] 氏名・社員番号・所属・勤務原価単価の登録
- [x] LocalStorage による設定永続化
- [x] 入力フォームへの自動反映（氏名・社員番号）
- [x] 全データ削除機能

## 経緯メモ

- **2026/04**: 時間外作業報告書アプリ（残業計算・Excel出力）初版リリース
