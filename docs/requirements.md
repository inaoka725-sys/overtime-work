# 要件定義書 — 時間外作業報告書アプリ

## 1. プロジェクト概要

### 成果目標
社員が時間外作業の実績を入力すると、規程に準拠した残業時間・深夜作業手当を自動計算し、提出用Excelファイルを出力する。

### 成功指標

**定量的指標**
- 残業手当の計算ミスをゼロにする
- 報告書作成時間を手書き・Excel手入力から 5分以内 に短縮
- 提出用Excelの出力まで追加作業なしで完結

**定性的指標**
- 規程（時間外手当基準）に完全準拠した計算が行われる
- 現場・社内どちらの作業区分にも対応できる
- 報告者が計算方法を知らなくても正しい手当額を把握できる

---

## 2. 規程準拠ルール

出典: `時間外手当基準（原紙）.xlsx`

### 残業開始時間
- **平日**: 常用勤務開始時間から **9時間30分後** から時間外手当を支給
- **休日**: 勤務開始時間から全時間が残業対象

### 休憩時間（規程定義）

| 時間帯 | 休憩 |
|--------|------|
| 08:30 〜 09:00 | 0.5h |
| 12:00 〜 13:00 | 1.0h |
| 15:00 〜 15:30 | 0.5h |
| 18:00 〜 18:30 | 0.5h |
| 22:00 〜 22:30 | 0.5h |
| 02:00 〜 03:00 | 1.0h |
| 06:00 〜 06:30 | 0.5h |

### 時間外手当の種別

| 種別 | 時間帯 | 金額 |
|------|--------|------|
| 普通残業手当 | 残業時間のうち 6:00〜22:00 | 給与計算による |
| 深夜残業手当 | 残業時間のうち 22:00〜翌6:00 | 給与計算による |
| 深夜作業手当 | 18:00〜翌6:00 の常用勤務が5時間以上の場合 | **一律 3,000円** |

**注意事項（規程より）**
- 深夜作業手当と同時間帯の時間外作業手当は重複して支給しない
- 時間外時間は 0.5時間単位で集計（四捨五入）

## 3. システム全体像

### 機能一覧

| # | 機能 | 概要 |
|---|------|------|
| F-001 | 作業入力 | 日付・氏名・オーダー番号・件名・区分・時間を入力 |

## 4. ページ詳細仕様

### P-001: 時間外作業報告書（単一ページ）

**目的**: 入力から Excel 出力まで一画面で完結

**入力フォーム**

| フィールド | 種別 | バリデーション |
|-----------|------|---------------|
| 日付 | date | 必須 |
| 報告者氏名 | text | 必須 |
| オーダー № | text | 任意 |
| 件名 | text | 任意 |
| 区分 | radio | 現場作業 / 社内作業 |
| 平日 / 休日 | radio | 平日 / 休日 |

| 帰社後作業 | checkbox + time×2 | 現場作業時のみ表示 |

**計算結果プレビュー（リアルタイム）**

- 普通残業 H

- 登録件数・普通残業合計・深夜残業合計・深夜作業手当合計
- 各エントリーの削除ボタン

**処理フロー**

```
入力
  ↓ リアルタイム計算（calcOvertime, calcLateNightWorkAllowance）

  ↓ 「この内容を追加」ボタン

  ↓ 「Excelファイルをダウンロード」ボタン

```

## 5. 計算ロジック詳細

### 残業時間算出

```
overtimeStart（平日）= 勤務開始時刻 + 570分（9時間30分）
overtimeStart（休日）= 勤務開始時刻

普通残業 = 残業時間帯 ∩ [6:00, 22:00] の実働時間（休憩除く）
深夜残業 = 残業時間帯 ∩ [22:00, 翌6:00] の実働時間（休憩除く）
```

### 深夜作業手当

```
常用勤務の深夜時間帯在籍 = [勤務開始, overtimeStart] ∩ [18:00, 翌6:00] の実働時間
深夜作業手当 = 常用勤務の深夜時間帯在籍 ≥ 300分（5時間）? 3,000円 : 0円
```

### 時間の丸め

0.5時間単位（四捨五入）: `Math.round(minutes / 30) * 0.5`

## 7. セキュリティ要件

- 静的ファイル配布のため、サーバーサイドセキュリティ要件なし
- 入力値はブラウザ内のみで処理し、外部送信しない
- SheetJS CDN の可用性に依存（オフライン利用時は事前キャッシュ推奨）

## 10. Excel出力仕様

### 報告書シート（個人別）

- ヘッダー: 時間外作業報告書、令和○年○月○日、報告者氏名
- 明細: 区分・平日休日・日付・オーダー №・件名・開始・終了・普通残業H・深夜残業H・深夜作業手当
- 合計行

### 一覧シート

- 全登録者の明細一覧（同カラム構成）
- 合計行

### ファイル名

`時間外作業報告書_YYYYMMDD.xlsx`
